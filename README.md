ShopAlpha is a full-stack ecommerce web application built as part of the CodeAlpha internship program. Users can register, log in, browse products with image galleries, add items to a cart with quantity selection, and complete checkout with order history. The admin panel allows authorized users to add, edit, and delete products with multiple images. The backend uses a REST API with JWT authentication, bcrypt password hashing, and a MySQL database. The frontend is built with React, React Router, Tailwind CSS, and shadcn/ui for a clean, responsive interface.
